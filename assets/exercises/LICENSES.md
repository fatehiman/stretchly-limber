# Exercise illustrations

Limber loads JPEG images from this folder by filename (referenced from
`config.json`). The defaults expect:

| Filename                        | Subject                                       |
| ------------------------------- | --------------------------------------------- |
| `eye_2020.jpg`                  | 20-20-20 eye break                            |
| `cervical_retraction.jpg`       | Chin tucks / cervical retraction              |
| `scapular_retraction.jpg`       | Shoulder rolls + scapular retraction          |
| `wrist_flexor_stretch.jpg`      | Wrist flexor / extensor stretch               |
| `lumbar_extension.jpg`          | Standing lumbar extension                     |
| `hip_flexor_stretch.jpg`        | Standing hip-flexor stretch                   |
| `walk_break.jpg`                | Stand & walk (full-rest)                      |

If a file is missing, the popup falls back to a text-only display (the
exercise title in place of the image). Drop your own JPEGs in here, or
sourced free / public-domain illustrations.

## Sourcing

Acceptable sources:

- **Public domain** — works whose copyright has expired or never applied
  (e.g. older NIH or OSHA publications).
- **CC0 / CC-BY / CC-BY-SA** — Creative Commons-licensed material;
  retain attribution per license terms in the table below.
- **Self-drawn** — simple line drawings authored for this project.

Not acceptable: copyrighted clinical illustrations, stock photos without
explicit redistribution rights, AI-generated medical imagery without
human review.
